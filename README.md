# KudEgOwo Product Documentation

Product information site for KudEgOwo beta testers — Schools, Parents, and Tutors.

**KudEgOwo** is a Family–School–Tutor Collaboration Platform that helps families, schools, and tutors work together seamlessly.

## Features Documented

- **School Payments** — Secure fee collection through trusted payment providers
- **SafeSchool** — Real-time attendance tracking and emergency alerts
- **Education** — Homework, grades, and digital report cards
- **Financial Literacy** — Teaching kids about money through fun lessons

## Development

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Deployment

This site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

**Live URL:** https://emergined.github.io/kdg-docs/
